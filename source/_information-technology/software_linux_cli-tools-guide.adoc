== Find

?Find directores , format output by using -exec & append to file using >> FileName


.Find directores , format output by using `-exec` & append to file using `>> FileName`
----
find DirName -name  "*.java"  

find "{pwd}" -maxdepth 1 -type d -exec echo "\"wpackagist-plugin/{}\" : \"*\" , " >> "_fileList.txt" \;

find  "{pwd}" -type f -exec "COMMAND" 
----
