= How to use a list to represent folder structure in CSS


----
<style>
.tree, .tree * {
font-size: 16px
}
.tree ul,
.tree li,
.tree ol
{
margin : 0px;
 list-style-type: none;
}

.tree * ul,
.tree * ol
{
margin-left : -10px;
}
.tree ul,
.tree ol {
border-left: 1px solid #444;
padding: 2px 10px;
}

.tree li {
    padding: 2px 10px;
    position: relative;
}
.tree li:before {
    content: "--";
    position: absolute;
    top 4px;
    left: -5px;
}
</style>
