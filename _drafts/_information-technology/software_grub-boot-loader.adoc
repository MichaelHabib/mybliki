= Grub Boot Loader

## Manually Installing grub2
```
bootfolder="/home/michael/media/multiboot_ext4/boot/"
bootdev="/dev/sda"
sudo grub-install $bootdev --root-directory=$bootfolder
sudo grub-mkconfig -o $bootfolder
```
